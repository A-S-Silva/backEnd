# EmplManagerBackEnd

Simple NodeJs backend servie for retrieving employees by position

## setup

The project requuires npm nodeJs installed

## Build Project

1. Install typescript: `npm install -g typescript`
2. Install npx: `npm install npx`
3. Install project dependecies: `npm install`

## Launch server

1. `npx ts-node src/server.ts`
2. Check loclahost ap port [8080](https://localhost:8080/)

## services:

| position | Url |
|------------ | -----
|All Employees  | https://localhost:8080/allemployees
|Juniors        | https://localhost:8080/juniors
